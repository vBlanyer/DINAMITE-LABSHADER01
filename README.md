## Descripción general

Se implementan tres shaders con ayuda de la libreria Three. El primero es un manipulador de vertices, el segundo es iluminación/sombreado Blinn-Phong y el tercero es uno creativo en el cual se implementa un toon shading y un efecto de globo. Para cada shader se implementan controles para modificar los parametros de cada uno, como por ejemplo la intensidad de la luz, el color del objeto, entre otros.

## Ejecucion 

Para arrancar el programa y observar los tres shaders desarrollados solo es necesario ejecutar los siguientes comandos.

```bash
npm install
npm run dev
```

Y colocar en el navegador el enlace localhost:5173, en esa interfaz sera sencillo pasar de un shader a otro por medio de los botones, como tambien jugar con los parametros de cada shader.
